Notes

On the radio version of the <input> tag, if two inputs have the same name="$name" attribute, they will be mutually exclusive.

Radio inputs will pass on the value in their value="$value" attribute.

form-check-inline can make your options be displayed inline, but all the options have to have the attribute.

The for attribute in the labels are supposed to link to the ids of items

When making buttons using the <input class="btn-check"> method, keep in mind that the label will mark the input that has the id included in the label's for="$id" attribute.

Yeah, I know that's confusing.
