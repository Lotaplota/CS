Notes

It's useful to be reminded that the ids referenced in the data-bs-targets should be preceeded by the #.

Accordions will behave weirdly if you don't add the collapse show bit to the class' parameters.

The buttons of collapsed items should be marked as collapsed to prevent them from looking active.

Add an id to the parent accordion and link all it's items via data-bs-parent="#id" to make the expansions exclusive.
