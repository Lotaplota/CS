Bordas e Sombras

Notes

To make an element have a border, add the border attribute. The border's parameters should be in other attributes.

The border attribute can be changed to border-bottom and vice versa to alter how the border is applied.

you only have shadow-sm and shadow-lg available, for some reason.

rounded-pill and rounded-circle are available as some other rounding presets, with some variations possible, like rounded-end-pill.

rounded-circle will make an ellipse if the element is a rectangle instead of a square.
