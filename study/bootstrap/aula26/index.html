Notes

You can opt for just container instead of container-fluid.

align-items-center should vertically align your items in a flex container.

The dropdown-toggle class attribute shows the little arrow on the right side of the item. It is not necessary.

When using bg-dark in the navbar, it was necessary to use navbar-dark alongside it.

Maybe this is necessary for good dark/light theme interaction.

dropdown-menu-end can be used to align the dropdown menu to end of the link.

py-$ can be used to make the navbar bigger.
