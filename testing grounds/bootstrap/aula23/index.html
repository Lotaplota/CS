Notes

Os elementos de um formulário ocupam 100% da largura do pai.

col divs work when inside a row div and their spacing is controlled by the class attribute g-$.

The class col-form-label allows you to choose a width for the label of a field. To do that, you must specify the width in the class attribute, col-2 for example.

col-form-label has sm and lg variants.
