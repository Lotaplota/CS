Notes

The <label> tag seems to display inline.

To do the pretty form thing, the <label> should be of class form-label, and the <input> should be of class form-control.

The disabled and readonly attributes unable the user to type in the form.

Using readonly alongside value="$value" allows the user to select and copy the $value.

form-control-sm and form-control-lg change the form's size.

plaintext makes the form borders disappear.

An <input>'s list attribute should be the id of a <datalist id="list-id"></datalist> declared somewhere.
