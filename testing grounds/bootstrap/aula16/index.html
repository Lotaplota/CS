For more information, watch aula 15 do curso de HTML e CSS from Ricardo Maroquio.

d-print-none pode ser utilizado para evitar que um elemento apareça na página de impressão.

Alternatively, d-print and it's variations allow the element to be shown only on printing.

The attributes visible and invisible control the visibility of an element, but maintain their space, unlike d-none.
